# Angular2 

Instalar:

- [NodeJS 7+](https://nodejs.org/)

- [Visual Studio Code 1.7+](https://code.visualstudio.com)

Desde un terminal, ejecutar:

~~~
$ npm install -g typescript@2.0
$ npm install -g typings
$ npm install -g angular-cli
$ ng new angular2-todolist
~~~
